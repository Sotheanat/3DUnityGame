🎮 Game Description
This project is a 2D pixel‑art fighting game built in Unity, featuring fast‑paced combat, retro visuals, and classic arcade‑style mechanics. Players control a martial‑arts fighter who can perform punches, kicks, and special energy attacks to defeat opponents in dynamic one‑on‑one battles.

✨ Key Features
Retro Pixel‑Art Style  
Characters, effects, and environments are designed with a nostalgic pixel aesthetic inspired by classic arcade fighters.

# Combat System

Punch attacks and kick attacks

Special moves with animated energy effects

Hit reactions, damage numbers, and health bar updates

Opponents react to player movement, take damage, and display health changes.

# Visual Effects

Energy beams

Charged power spheres

Impact flashes

Smooth sprite animations
